# ProntApp

Frontend desarrollado con **Next.js**, **React**, **TypeScript** y **Tailwind CSS**, enfocado en rendimiento, escalabilidad y experiencia de usuario moderna.

## Stack

* Next.js 14
* React 18
* TypeScript
* Tailwind CSS
* Zustand
* PostgreSQL

## Instalación

```bash
git clone https://github.com/luchomdev/prontapp.git
cd prontapp
npm install
npm run dev
```

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Características

* Arquitectura moderna con Next.js
* Estilos con Tailwind CSS
* Estado global con Zustand
* Integración con PostgreSQL
* Edición enriquecida de contenido
* Gráficos y visualización de datos
* Soporte para notificaciones push
